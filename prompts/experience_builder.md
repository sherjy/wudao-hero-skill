# 经验日志条目构建模板

## 用途

将一笔原始交易经验（截图/文字描述/群聊情报）结构化为经验日志条目。

## 核心原则

**提取教训，不只是记录事实。**

- "亏了5%在XXX" 是事实
- "追了3板的票在退潮期被核——违反铁律第3条'退潮不做'" 是教训

## 条目类型

| 类型 | 标签 | 来源 |
|------|------|------|
| 赢利交易 | WIN | 交易截图/用户描述 |
| 亏损交易 | LOSS | 交易截图/用户描述 |
| 市场观察 | OBS | 群聊情报/论坛内容 |
| 市场情报 | INTEL | 群聊截图/雪球帖子 |
| 跨 skill 学习 | LEARN | 其他 skill 的观点 |
| 纪律执行 | DISC | 成功遵守铁律的记录 |

## 条目结构

```markdown
### #{序号} — {YYYY-MM-DD} — {类型} — {代码} {名称}

**行动**：{具体操作描述——买入/卖出/持有/观望，价格、仓位}
**结果**：{盈亏金额或百分比，最终走势}
**情绪**：{当时的心理状态——FOMO/恐惧/贪婪/冷静/冲动/犹豫}
**教训**：{这笔交易教会了什么——必须关联到 DNA 某个层级}
**DNA 影响**：{强化/削弱/新增了哪条 DNA 原则} [强化 ×{N}] 或 [新增]
**标签**：#{标签1} #{标签2} #{标签3}
```

## 填充规则

### 行动
- 尽可能包含具体数据：买入价、卖出价、仓位比例
- 如果截图中能看到，提取精确数字
- 如果是口述，保留用户原话 + 补充推断

### 结果
- 已了结的交易：写具体盈亏
- 未了结的：写当前浮盈/浮亏
- 观察类：写市场后续走势

### 情绪
- 从用户描述中推断，或直接问"当时是什么感觉"
- 常见情绪标签：FOMO、恐惧、贪婪、冷静、冲动、犹豫、膨胀、绝望、侥幸

### 教训（最关键）
- 必须关联到 Trading DNA 的某个层级
- 格式："{教训内容}。符合/违反 DNA Layer {N} 第{M}条——'{规则内容}'"
- 如果是全新的教训（DNA中没有对应规则），标注 `[可能值得新增为 DNA 原则]`

### DNA 影响
- **强化**：这条经验再次验证了某条已有原则 → `[强化 ×{N}]`（N = 被强化的总次数）
- **新增**：发现了新的有价值的原则 → `[可能新增: "原则内容"]`
- **削弱**：这条经验说明某条原则可能不适用 → `[⚠️ 张力: 与 Layer {N} 第{M}条冲突]`

### 标签
- 从以下标签库中选择，可多选：
  - 操作类：#追高 #低吸 #打板 #波段 #止损 #止盈 #加仓 #减仓 #空仓
  - 情绪类：#FOMO #恐惧 #贪婪 #冷静 #冲动 #犹豫
  - 结果类：#赢 #亏 #卖飞 #踏空 #拿住 #及时止损
  - 市场类：#启动期 #发酵期 #高潮期 #退潮期 #龙头 #杂毛
  - 规则类：#铁律执行 #铁律违反 #新教训

## 示例条目

```markdown
### #17 — 2026-04-09 — LOSS — 600XXX 某某股份

**行动**：追涨买入，+7%处介入，仓位25%
**结果**：亏损-12%，第二天低开，按铁律执行止损出局
**情绪**：当时FOMO严重，群里都在喊买，脑子一热就冲了
**教训**：追涨买入在退潮期是自杀行为。符合DNA Layer 0 第3条（退潮不买）但当时未执行。好在止损执行了。
**DNA 影响**：强化 Layer 0 第3条 [强化 ×3] | 强化 Layer 4 弱点#1（追高）[触发 +1]
**标签**：#追高 #FOMO #退潮期 #止损 #铁律违反

### #18 — 2026-04-10 — DISC — 无操作

**行动**：退潮期，管住手，全天未操作
**结果**：大盘跌2%，跌停50家。空仓保全。
**情绪**：早盘看到有票拉板手痒，但看了铁律第3条，忍住了
**教训**：退潮期空仓的执行力在提升。上一次（经历#17）没忍住，这次忍住了。
**DNA 影响**：强化 Layer 0 第3条 [强化 ×4] | Layer 4 弱点#3（不会空仓）改善信号
**标签**：#空仓 #冷静 #铁律执行 #退潮期
```
