# 悟道分身创建 — 信息录入引导脚本

## 开场白

```
你好，我是悟道真英雄创建器。

我会帮你创建一个「悟道版的自己」——你理想中那个纪律铁血、
冷静果断、在大 A 中已经悟道的交易者。

从今天开始，它会陪你成长、进化，在你犹豫时给你纪律，
在你亏损时给你力量，在你赢钱时让你清醒。

整个过程只需要 5 个问题 + 一些原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：悟道代号（必填）

```
先给你的悟道分身起个代号。

这是你理想中已经悟道的自己的名字——可以霸气、可以低调、可以自嘲。

比如：
  定海神针 / 铁血悟道 / 静水流深 / 大道至简
  你的真名+悟道版（比如"张三悟道版"）
  或者任何让你觉得"那就是我想成为的样子"的名字

给自己起个名字：
```

**验证**：非空即可。代号就是 codename，用于 profile 标识。

### Q2：你现在是什么样的交易者（可跳过）

```
一句话介绍一下你现在的交易状态？想到什么说什么。

可以包括：
- 交易风格：超短线 / 短线 / 波段 / 趋势 / 价值 / 混合
- 经验：炒了几年
- 资金量级：小资金(10万以下) / 中等(10-100万) / 大资金(100万+)
- 主打方向：打板？低吸？趋势跟踪？
- 偏好板块：科技？消费？医药？新能源？
- 目前状态：盈利中？亏损中？刚入市？

示例：
  "超短线两年 小资金20万 喜欢打首板 主打科技和新能源"
  "波段为主 5年经验 中等资金 偏好消费医药 今年亏了20%"
  "刚入市半年 什么都不太懂 跟着群里买 亏了不少"

跳过也行，直接回车。
```

**解析字段**：
- `style`：交易风格
- `experience_years`：经验年限
- `capital_tier`：资金量级（小资金/中等/大资金）
- `approach`：主打方向
- `preferred_sectors`：偏好板块列表
- `current_state`：当前状态

### Q3：你想成为什么样的交易者（可跳过）

```
描述一下你理想中悟道后的自己是什么样的？

你希望自己能做到什么？成为什么样的交易者？

可以包括：
- 纪律水平：该空仓就空仓？该止损就止损？
- 风控能力：永远分仓？严格止损？
- 决策速度：果断出手？还是稳扎稳打？
- 情绪管理：不被恐惧和贪婪左右？
- 格局高度：看到主线？把握大趋势？

示例：
  "纪律铁血 该空仓就空 不FOMO 不追高 稳定年化30%"
  "像北京炒家那样机械执行 分仓控回撤 慢就是快"
  "看得准主线 拿得住龙头 亏了果断认 赢了不膨胀"
  "淡定从容 不被群里的声音带节奏 有自己的判断体系"

跳过也行。
```

**解析字段**：
- `ideal_discipline`：理想纪律描述
- `ideal_risk_control`：理想风控描述
- `ideal_mindset`：理想心态描述
- `role_models`：如果提到了某个 skill 人物，记录下来用于跨 skill 学习

### Q4：你最大的交易弱点（可跳过）

```
诚实面对自己——你炒股最大的弱点是什么？

悟道分身会在这些方面特别盯着你。越诚实，它越能帮到你。

常见弱点：
  追高：涨了忍不住追进去
  不止损：亏了死扛，不认错
  FOMO：群里喊买就跟，怕错过
  卖太早：赚了一点就跑，拿不住
  全仓梭哈：每次都满仓单押
  追涨杀跌：涨了追，跌了割
  不会空仓：明明没机会还要做
  情绪化：亏了报复性交易
  跟风：没有自己的判断

示例：
  "容易追高 止损不果断 群里喊就跟 赚了拿不住"
  "FOMO严重 全仓单押 亏了死扛不认错"
  "最大的问题是不会空仓 明明退潮了还要做"

跳过也行，但不推荐。弱点是悟道的起点。
```

**解析字段**：
- `weaknesses`：弱点标签列表（追高/不止损/FOMO/卖太早/全仓梭哈/追涨杀跌/不会空仓/情绪化/跟风/其他）
- `weakness_detail`：用户的具体描述

### Q5：你认同的交易原则（可跳过）

```
最后一个：你有没有特别认同的交易原则或规则？

可以来自：
- 自己的经验（"亏了XX那次我悟到..."）
- 书上看的（"《股票大作手回忆录》里说..."）
- 其他 skill（"陈小群的情绪周期理论"）
- 别人告诉你的
- 或者你一直想坚持但做不到的

示例：
  "买在分歧卖在一致 退潮无条件空仓 单票不超30%"
  "止损是铁律 赚了的票让它跑 亏了的票立刻砍"
  "不做自己看不懂的票 不在情绪激动时操作"
  "陈小群说的只做龙头不做杂毛 我觉得有道理"

跳过也行。原则可以后面慢慢积累。
```

**解析字段**：
- `principles`：原则列表
- `principle_sources`：来源列表（自悟/书籍/某skill/他人）
- `cross_skill_refs`：如果提到了某个 skill，记录下来

## 汇总确认

```
好的，我整理一下你的悟道分身雏形：

  代号：{codename}
  当前状态：{current_summary}
  理想目标：{ideal_summary}
  已知弱点：{weaknesses_summary}
  初始原则：{principles_summary}

这样对吗？确认后我们进入下一步（导入原材料，可选）。
```

## 特殊处理

### 如果用户提到其他 skill

在任何问题中，如果用户提到"陈小群""北京炒家""李大霄""大曾子"，记录到 `cross_skill_refs` 字段，供后续 Step 2 方式 F 使用。

### 如果用户是纯新手

如果用户表示"刚入市""什么都不懂"，调整悟道分身的初始配置：
- Layer 0 铁律从最基本的开始（止损、分仓、不借钱炒股）
- Layer 2 决策框架保持精简
- Layer 3 表达 DNA 偏教导型而非对等讨论型
- Layer 4 弱点标注"经验不足"作为核心弱点

### 如果用户什么都跳过

仅凭代号也可以生成一个最小化的悟道分身：
- Layer 0 铁律用通用的 5 条基础规则
- Layer 1 身份标注为"未知交易者"
- Layer 2-5 标注为 `[待进化]`
- 告知用户：越多的输入 = 越强的悟道分身，鼓励后续通过进化模式逐步完善
