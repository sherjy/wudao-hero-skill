# 炒股群聊截图分析维度

## 用途

当用户提交微信/QQ 炒股群聊天截图或聊天记录导出文件时，按以下维度提取市场情报。

## 分析目标

从群聊中提取**市场情报**，而非个人性格（与 ex-skill 的 chat_analyzer 不同）。

## 提取维度

### 1. 股票提及

```markdown
**提及的股票**：
| 代码/名称 | 提及次数 | 主流观点 | 情绪 |
|----------|---------|---------|------|
| {ticker} | {count} | {看多/看空/分歧} | {兴奋/恐慌/冷静} |
```

**识别规则**：
- 6 位数字代码（600xxx, 000xxx, 300xxx, 688xxx）
- 公司简称
- 板块/概念名称（如"AI""新能源""芯片"）

### 2. 群内情绪

```markdown
**群内情绪判断**：
- 整体情绪：{极度兴奋/乐观/中性/悲观/恐慌}
- 一致性：{高度一致/有分歧/严重分裂}
- FOMO 信号：{有/无}（表现：{具体描述}）
- 恐慌信号：{有/无}（表现：{具体描述}）
```

**情绪关键词**：
- 兴奋/FOMO：冲冲冲、上车、起飞了、牛逼、加仓、梭哈、明天继续涨
- 恐慌：完了、跑了、割了、崩了、跌停、凉了、核按钮
- 分歧：不看好、但是、风险、小心、我觉得不行

### 3. 有价值的观点

```markdown
**值得关注的观点**：
1. {发言者}：{观点内容}
   - 逻辑质量：{高/中/低}
   - 是否有数据支撑：{是/否}
   - 与用户 DNA 的相关性：{高/中/低}
```

### 4. 警告信号

```markdown
**风险警告**：
- 羊群效应：{全群看多且无反对声音 = 高风险信号}
- 情绪顶点：{过度兴奋可能意味着阶段顶部}
- 恐慌底部：{过度恐慌可能意味着机会，但需结合 DNA 规则判断}
```

## 与 DNA 的关联

提取完毕后，对照用户的 Trading DNA 给出建议：

1. 如果群内一致看多，提醒用户 DNA 中是否有"买在分歧卖在一致"的规则
2. 如果群内恐慌，提醒用户是否有逆向操作的框架
3. 如果发现 FOMO 信号，关联用户 Layer 4 弱点中的"FOMO"或"跟风"

## 输出要求

- 提取的是**市场情报**，不是对群友的性格分析
- 重点标注与用户 DNA 相关的信号
- 明确标注哪些是"事实"（群里说了什么），哪些是"判断"（AI 的情绪分析）
